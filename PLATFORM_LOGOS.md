- **Logo download**: https://brandfetch.com/goto.com
- **Alternative**: Search "GoToWebinar logo SVG" on Google Images with usage rights filter

## 2. EverWebinar

- **Alternative**: https://brandfetch.com/everwebinar.com

## 3. WebinarJam

- **Alternative**: https://brandfetch.com/webinarjam.com

## 4. Google Sheets
- **Official site**: https://workspace.google.com/products/sheets/
- **Brand resources**: https://about.google/brand-resource-center/
- **Direct**: Use the official Google Sheets icon from Google's design resources

## 5. DonorLead
- **Official site**: https://donorlead.net
- **Logo location**: Header logo on homepage
- **Alternative**: Contact DonorLead support for logo assets

## Recommended Approach

1. **Option 1** (Fastest): Use brandfetch.com to download logos in PNG/SVG format
   - Go to brandfetch.com and search for each platform
   - Download the logo in PNG format (preferably transparent background)
   - Resize to approximately 200x200px or similar square dimensions

2. **Option 2** (Official): Visit each platform's brand/press page
   - Most companies have a "Brand Assets" or "Press Kit" page
   - Download official logos with proper usage rights

3. **Option 3** (Manual): Screenshot and crop from official websites
   - Visit the platform's homepage
   - Screenshot the header logo
   - Crop and remove background using an image editor

## File Naming Convention

All files should be named: `{slug}.png` where slug matches the platform slug in platform.json

Examples:
- `gotowebinar.png`
- `everwebinar.png`
- `webinarjam.png`
- `google_sheets.png`
- `donorlead.png`

Place all files in: `apps/web/public/images/platforms/`
